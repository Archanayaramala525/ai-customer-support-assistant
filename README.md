# AI Customer Support Assistant

An AI-powered customer support chatbot that answers questions using company documents.

This project uses **Retrieval Augmented Generation (RAG)** to search documents and generate intelligent responses.

## Technologies Used

- Python
- LangChain
- HuggingFace Transformers
- FAISS Vector Database
- FastAPI
- Docker (optional)

## Features

- AI-powered document search
- Customer support automation
- Semantic search using embeddings
- REST API using FastAPI
- Scalable AI architecture
